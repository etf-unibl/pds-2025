Detailed Description

Look-ahead buffer architecture for the preamble_generator.

Implements a Moore-type finite state machine (FSM) that outputs an alternating 1/0 preamble sequence ("10101010"). The architecture uses:

  • A state register (state_reg)
  • A next-state FSM (state_next)
  • A one-cycle look-ahead output buffer (buf_reg) The FSM transitions through a fixed sequence of states once start_i is asserted. Each state corresponds to a specific output bit.
